## 🗂️ 考点清单（17 类）

| 考点 | 关键内容 |
|---|---|
| 驾照制度 | 记点（7 点上课减分 / 12 点吊照）、拒测、恢复驾照 |
| 酒精药物 | BAC 0.05% 受损 / 0.08% 醉、肝脏代谢、酒+药互相增强 |
| 行人自行车 | 走路逆行、骑车顺行；盲人白杖/导盲犬绝对优先 |
| 限速 | 市区 35 / 市外 55 / 校车 45 / 课外活动车 55 |
| 转弯车道 | 弯前减速、弯中不刹车、等左转车轮摆直 |
| 超车 | 双黄实线禁超、看到被超车保险杠再回线 |
| 路权 | 先到先行、同到让右、行人始终优先 |
| 安全驾驶 | 跟车两秒、勤看后视镜、任何情况系安全带 |
| 疲劳 / 夜间 / 恶劣天气 | 困倦只能休息；市区雾天用近光；刚下雨最滑 |
| 紧急故障 | 刹车失灵换低档+手刹；爆胎松油门稳方向；落水开窗逃 |
| 高速公路 | 最常见追尾；加速车道；错过出口到下个出口 |
| 车辆保养 | 每年验车、必备牌照灯/安全带/喇叭 |
| 交通信号 | 红闪=停、黄闪=减速、菱形=警告、施工橙色 |
| 校车平交道 / 停车规定 | 同向必停校车；消防栓旁 15 尺；多处禁停区 |

> 进入网页点击「📚 考点总结」可看到完整的速记清单。
