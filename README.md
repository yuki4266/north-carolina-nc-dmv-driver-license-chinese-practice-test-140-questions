## ✨ 功能特性

### 📖 题目与解析
- **140 道完整考题**，覆盖 **17 大考点**（见下方清单）。
- **逐题详细解析**，统一结构：
  **题意背景 → 逐个选项为什么对 / 为什么错 → 具体法规与数字 → 💡 记忆口诀 → 🔗 关联题**。
- **数字记忆题配「搞笑口诀」**，越损越好记（限速、跟车距离、罚款、记点、距离换算等）。
- **英制 → 公制换算**：解析里把「尺 / 英尺」换算成「米」，方便习惯公制的人。

### 🎯 交互做题
- **点选项即时判对错**：选对变绿、选错变红，并自动展开答案与解析。
- **一键展开 / 收起** 全部答案；可按**考点筛选**、**关键词搜索**。

### 🧪 测验 / 刷题模式
- 可选范围：**全部 140 题 / 随机 30 / 随机 50 / 按考点出题 / 只做错题本**。
- 做完一轮自动出 **成绩单**：得分百分比、**按错题数排出你最易错的考点 + 对应知识点速记**、错题清单（可点击跳转复习）。
- **一键重做本轮错题**，循环练到全对。

### 📕 错题本 & 📊 进度
- 答错的题**自动收进错题本**，随时重做；答对后自动移出；可一键清空。
- **置顶进度条**实时显示「已完成 X / 140 题 · 百分比」。
- 错题与进度**本地保存**（浏览器 `localStorage`），关掉再打开依然在。

### 🚦 交通标志图解
- 重新绘制的**清晰矢量版**交通标志速查（形状 / 颜色与真实标志一致，**缩放不糊**），约 40 个标志，每个配**中文名称 + 详细说明**，分 6 大类。

### 💾 纯本地、零依赖
- **单个 HTML 文件**，所有图片以 base64 内嵌；**无后端、无数据库、无第三方追踪、不联网**也能用。
- 浅色清爽 UI，**响应式设计**，手机端体验良好。

## 🧪 测验模式详解

1. 点顶部 **📝 测验 / 刷题**。
2. 选范围：`全部 140 题` / `随机 30` / `随机 50` / `按考点` / `只做错题本`。
3. 逐题作答，底部浮动条实时显示 **已答 x / 总数 · ✅ 对 · ❌ 错**。
4. 做完点 **「本轮总结」**，看到：
   - 🏆 **得分**（绿 ≥80% / 橙 ≥60% / 红 <60%）
   - 🎯 **最易错考点排行** + 每个考点的知识点速记
   - 📕 **错题清单**（点题号跳转复习）
5. 点 **「重做这 N 道错题」** → 只刷错题，**循环到全对** 🎉。

## 🔁 推荐复习法（错题循环）

> 亲测高效的"三轮滚动法"：

1. **第一轮**：`全部题目` 模式从头过一遍，点选项即时看解析，重在**理解**。
2. **第二轮**：`📝 测验 → 全部 140 题`，模拟考一遍，做完看**易错考点总结**。
3. **第三轮起**：`📝 测验 → 只做错题本`，反复 **「重做错题」**，直到错题本清空。
4. 考前一晚：扫一遍 **📚 考点总结** + **🚦 标志图解**，强化数字和标志记忆。

## 🗂️ 考点清单（17 类）

| 考点 | 关键内容（含数字速记） |
|---|---|
| 驾照制度 | 记点：7 点上课减分 / 12 点吊照；拒测吊 12 个月；恢复驾照需许可+恢复费 |
| 酒精药物 | BAC 0.05% 受损 / 0.08% 醉；肝脏代谢酒精；酒+药互相增强；未成年零容忍吊 12 个月 |
| 行人自行车 | 走路逆行（面对车）、骑车顺行；盲人白杖/导盲犬绝对优先；骑车意外多在住宅区 |
| 限速 | 市区 35 / 市外 55 / 校车 45 / 课外活动车 55；安全车速看天气路况 |
| 转弯车道 | 弯前减速、弯中不刹车（车甩外侧）；右转贴右；等左转把车轮摆直 |
| 超车 | 双黄实线禁超；前车让行人时禁超；看到被超车保险杠再回线；被超时略减速 |
| 路权 | 无控制路口：先到先行、同到让右、行人始终优先；从私车道驶出要让 |
| 安全驾驶 | 跟车两秒（雨天加倍）；勤看后视镜防追尾；任何情况都系安全带 |
| 疲劳驾驶 | 困倦只能停车休息；眼睛不停扫描、别盯一处 |
| 夜间驾驶 | 市区/雾天用近光；跟车会车禁远光；眩光看右侧白线 |
| 恶劣天气 | 刚下雨最滑、车距加倍；湿刹车低档轻踩烘干；打滑转向车尾方向+松油门 |
| 紧急故障 | 刹车失灵换低档+手刹；爆胎松油门稳方向勿急刹；落水开窗逃；抛锚支盖+白布+双闪 |
| 高速公路 | 最常见追尾；太慢也危险；加速车道并入；错过出口到下个出口 |
| 车辆保养 | 每 12 个月验车；必备牌照灯/安全带/喇叭；排气漏=毒气进车 |
| 交通信号 | 红闪=停、黄闪=减速；菱形=警告；施工橙色；警察指挥高于信号 |
| 校车平交道 | 多车道同向必停校车；过铁道口停车看两边 |
| 停车规定 | 消防栓旁 ≥15 尺；桥上/路口/平交道 50 尺内/消防栓前禁停 |

> 进入网页点击 **📚 考点总结** 可看到完整的可折叠速记清单。

## ❓ 常见问题 FAQ

**Q：需要登录或联网吗？**
A：都不需要。纯前端，断网也能刷；进度存在你自己的浏览器。

**Q：换了浏览器 / 清了缓存，进度还在吗？**
A：进度和错题存在当前浏览器的 `localStorage`，**换设备或清缓存会丢失**。

**Q：手机能用吗？**
A：能，响应式设计；可"添加到主屏幕"当 App 用。

**Q：答案可靠吗？**
A：已与原卷答案表逐题核对（140/140 一致）；个别有争议的原题已 ⚠ 标注。

**Q：内容会随官方更新吗？**
A：题库为固定备考资料，**请同时以 NCDMV 官方最新《驾驶手册》为准**。
